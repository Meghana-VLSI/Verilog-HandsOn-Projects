
# Combinational Circuits

This folder contains Verilog implementations of basic and advanced combinational logic designs using Xilinx Vivado.

## Included Designs

- ALU
- Barrel Shifter
- Binary to Gray Code Converter
- Comparator
- 2-to-4 Decoder
- Demultiplexer
- 8-to-3 Encoder
- Full Adder
- Full Subtractor
- Half Adder
- Half Subtractor
- 8-to-1 Multiplexer
- Parity Function
- Parity Generator
- Seven Segment Display

Each project includes:
- Verilog Design File
- Testbench
- Simulation Output
- schematic
- Synthesis Output

Tool Used:
- Xilinx Vivado
